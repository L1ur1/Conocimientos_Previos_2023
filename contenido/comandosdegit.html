COMANDOS DE GIT



  • git init:
    Inicializa un nuevo repositorio.
  • git add:
    Añade cambios al área de preparación.
  • git commit:
    Guarda los cambios en el repositorio.
  • git status:
    Muestra el estado de los archivos en el directorio de trabajo.
  • git log:
    Muestra un historial de confirmaciones.
  • git branch:
    Lista, crea o borra ramas.
  • git checkout:
    Cambia de rama o restaura archivos.
  • git merge:
    Combina cambios de diferentes ramas.
  • git remote:
    Muestra nombres de los repositorios remotos.
  • git fetch:
    Descarga objetos y referencias de otro repositorio.
  • git pull:
    Obtiene cambios del repositorio remoto e integra en el repositorio local.
  • git push:
    Actualiza referencias remotas junto con objetos asociados.
  • git clone:
    Clona un repositorio en un nuevo directorio.
  • git diff:
    Muestra cambios entre commits, commits y árbol de trabajo, etc.
  • git reset:
    Restablece la posición HEAD actual a la rama especificada.
  • git rm:
    Elimina archivos del directorio de trabajo y del índice.
  • git mv:
    Mueve o renombra un archivo, directorio o enlace simbólico.
  • git grep:
    Muestra líneas que coinciden con un patrón de búsqueda.
  • git stash:
    Guarda temporalmente cambios que no están listos para ser confirmados.
  • git blame:
    Muestra quién modificó por última vez cada línea de un archivo y cuándo.
