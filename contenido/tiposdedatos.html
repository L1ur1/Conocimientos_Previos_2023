TIPOS DE DATOS

NÚMEROS

Si en la base de datos se necesita almacenar números con o sin decimales, se debe pensar en un tipo

TIPO DE DATO TAMAÑO SINTAXIS
TINYINT 0 a 255 edad tinyint
SMALL INT 0 a 65535 Puntos small int
MEDIUM INT 0 a 16777215 Valor_Articulo medium int
INT 0 a 4294967215. A este tipo de campo se le puede declarar un ancho. Valor_Venta int (8)
BIG INT 0 a 18,446 * 1036
FLOAT(M,D) Estos tipos de datos son utilizados para decimales, el tamaño varía según los parámetros.
M: Número de la Mantisa
D: Número de Decimales		 Definitiva_materia float (2,1)
DOUBLE(M,D) Saldo_cuenta double
DECIMAL(M,D)
