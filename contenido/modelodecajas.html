MODELO DE CAJAS

El modelo de cajas en CSS describe la estructura de un elemento en términos de su contenido, padding, borde y margen. Esto permite controlar el diseño y el espacio que ocupa un elemento en una página web.



  • width:
    Ancho del contenido de un elemento.
  • height:
    Altura del contenido de un elemento.
  • margin:
    Espacio externo alrededor de un elemento.
  • padding:
    Espacio interno alrededor del contenido de un elemento.
  • border-box:
    Modelo de caja que incluye el ancho y alto del contenido y el padding, pero no el margen.
  • content-box:
    Modelo de caja que incluye solo el ancho y alto del contenido.
