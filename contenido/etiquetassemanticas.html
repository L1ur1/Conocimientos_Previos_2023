ETIQUETAS SEMÁNTICAS

Las etiquetas HTML semánticas son etiquetas que definen el significado del contenido que engloban.



  • Header:
    Define el encabezado de una página web o sección.


  • Nav:
    Define una sección con enlaces de navegación.


  • Section:
    Define una sección dentro de una página web.


  • Article:
    Define un bloque de contenido autónomo que puede existir en cualquier contexto.
    Puede tener su propio encabezado, pie de página, secciones... Útil para una lista de publicaciones del blog.


  • Aside:
    Define un bloque de contenido que está relacionado con el contenido principal.
    Generalmente se muestra como una barra lateral.


  • Footer:
    Define el pie de página de una página o sección web.


  • Main:
    Define el contenido principal de un documento.
