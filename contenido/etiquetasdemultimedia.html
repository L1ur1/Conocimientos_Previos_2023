ETIQUETAS MULTIMEDIA

Las etiquetas que corresponden al grupo de imagen y multimedia en HTML son los encargados de incrustar y controlar cualquiera de los elementos multimedia o de imágenes que quieres emplear para el desarrollo de tu app o programa web.



  • Audio:
    Define un elemento de audio.


  • img:
    Define una imagen.


  • Vídeo:
    Define un elemento de vídeo.


  • Source:
    Define múltiples recursos multimedia para elementos de audio y vídeo.
