JAVASCRIPT



  • Variables:
    Almacenan datos que pueden ser modificados durante la ejecución. Pueden declararse utilizando las palabras clave var, let o const.
  • Tipos de datos:
    Incluyen números (enteros y decimales), cadenas (texto), booleanos (true o false), objetos (estructuras de datos complejas), arrays (listas de elementos) y undefined (cuando una variable no tiene un valor asignado).
  • Operadores:
    Permiten realizar operaciones aritméticas (suma, resta, multiplicación, división), lógicas (and, or, not) y de comparación (igualdad, desigualdad, mayor que, menor que).
  • Condicionales:
    Estructuras como if, else if y else para tomar decisiones basadas en condiciones lógicas.
  • Bucles:
    Estructuras como for y while para la repetición de código. Permiten ejecutar un bloque de código varias veces mientras se cumpla una condición específica.
  • Funciones:
    Bloques de código reutilizables que realizan una tarea específica. Pueden recibir argumentos y devolver un valor.
  • Objetos:
    Agrupan propiedades (características) y métodos (acciones) relacionados. Los objetos en JavaScript pueden ser predefinidos (como Math o Date) o definidos por el usuario.
  • Eventos:
    Interacciones del usuario que desencadenan acciones en el código. Los eventos pueden ser de diferentes tipos, como clics de ratón, pulsaciones de teclado, carga de página, entre otros.
  • DOM (Document Object Model):
    Representación del documento HTML que JavaScript puede manipular. Permite acceder, modificar y eliminar elementos y contenido de una página web de forma dinámica.
  • AJAX (Asynchronous JavaScript and XML):
    Permite realizar solicitudes asíncronas al servidor web sin necesidad de recargar la página completa. Se utiliza principalmente para cargar datos de forma dinámica y actualizar partes específicas de una página.
