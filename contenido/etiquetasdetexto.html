ETIQUETAS DE TEXTO

Las etiquetas de texto son las que dan significado a los textos que contienen y permiten modificarlos. Las etiquetas de texto en HTML son aquellas que permiten introducir texto y modificar su formato de texto en html, de manera que se adapte a los deseos e intereses de cada desarrollador web.



  • < p >:
    Define para escribir párrafos.


  • < h1 > - < h6 >:
    Define títulos o encabezados de nivel 1 al nivel 6.


  • < span >:
    Define un fragmento de texto en línea.


  • < strong >:
    Define texto importante, usualmente representado en negrita.


  • < em >:
    Define texto enfatizado, usualmente representado en cursiva.


  • < abbr >:
    Define una abreviatura o acrónimo.


  • < cite >:
    Define una referencia a una obra


  • < code >:
    Define código de computadora.


  • < pre >:
    Define texto preformateado.


  • < blockquote >:
    Define una cita larga de otro autor.


  • < q >:
    Define una cita corta en línea.


  • < a >:
    Define un enlace.
